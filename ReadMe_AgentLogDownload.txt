Script will get executed on any windows/linux instance. It will download the agent logs for the instance as per given Instance Id.

Pre-requisites :

Python 2.7.15 or higher version should be installed

pip should be installed

Following packages/modules should be installed :
requests
json
time
sys
re
os

How to run :

python downloadagentlogs.py <Customer ID> <Domain ID> <Client Id> <Client Secret Key> <Instance ID>
